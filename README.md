# 💬 Toxic Comment Detector

This is a Machine Learning-based web app that detects toxic comments.

## 🚀 Features
- Detects toxic vs non-toxic comments
- Uses trained ML model
- Built using Flask

## 🛠️ Tech Stack
- Python
- Flask
- Scikit-learn

## ▶️ Run
pip install flask numpy pandas scikit-learn  
python app.py
